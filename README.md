## 🛠️ Tecnologias e conceitos
- Versionamento de código

O Versionamento de código veio para facilitar a vida de empresas e progamadores. O versionamento permitiu o registro de mudanças em arquivos, que possibilita a recuperação de versões anterioes, com isso, evitando diversos prejuizos em ambientes de trabalho. Além disso, permitiu um desenvolvimento de código em colaboração com integrantes.

- Git 

O Git é um sistema de versionamento de código, que guarda os registros de versão snapshots (fotos) do estado do projeto, além do caminho que o projeto percorre. O git é um conjunto de softwares que realiza esses versionamentos com bastante precisão, suas operações em grande parte são locais  e por isso boa parte das operações são praticamente instantâneas.

# 🛠️ Comandos Git 
## Comando: para iniciar sua jornada no Git

git config --global user.name "seu nome"

git config --global user.email "seu melhor email"

Estes comandos serviram para cadastrar o usuário que está realizando os versionamentos.

## Como acessar a um repositório git ?

## Podemos criar de duas formas: de forma remota ou de forma local

De forma forma remota:

==> Se você estiver utilizando o GitHub, basta entrar na interface do seu repositório, clicar em "Code" e copiar o link do "HTTPS";

==> Depois você deve criar uma pasta em seus arquivos onde ira armazenar esse repositório;

==> Você deve copiar o caminho dessa pasta e abrir o Prompt de comando do seu computador;

==> Logo depois você vai executar os seguintes comandos:

cd caminho-da-sua-pastar ==> este comando vai servir para você entrar na pasta

git clone -link-do-seu-repositório-remoto ==> este comando vai adicionar seu repositório na pasta que você entrou

De forma Local:

==> Você vai abrir a pasta onde vai iniciar seu projeto e vai abrir essa pasta na sua IDE de preferencia;

==> Depois você vai utilizar o terminal da sua IDE e realizar o seguinte comando:

git init

==> Dessa forma ele inicia um repositório de forma local, é possivel realizar esse mesmo comando pelo Cmd
